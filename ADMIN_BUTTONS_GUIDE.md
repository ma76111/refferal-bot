# 👨‍💼 دليل أزرار الأدمن - شرح مبسط

## 🎛️ لوحة التحكم الرئيسية

عندما تضغط على **⚙️ لوحة التحكم**، تظهر لك هذه الأزرار:

---

## 1️⃣ ✅ مراجعة المهام

**وظيفته:**
- يعرض لك كل الإثباتات اللي المستخدمين بعتوها
- تقدر تقبل أو ترفض كل إثبات

**كيف يشتغل:**
1. تضغط على الزر
2. يظهر لك عدد الإثباتات المعلقة
3. لكل إثبات يظهر:
   - اسم المستخدم
   - المهمة اللي عملها
   - الإثبات (نص أو صور)
   - زرارين: ✅ قبول | ❌ رفض

**مثال:**
```
📋 عدد المهام قيد المراجعة: 5

🆔 123 - محمد
🤖 المهمة: اشترك في قناة ABC
📝 الإثبات: "تم الاشتراك"
📸 [صورة]

[✅ قبول] [❌ رفض]
```

**لما تقبل:**
- المستخدم ياخد المكافأة
- المهمة تتحسب مكتملة

**لما ترفض:**
- تختار: رفض عادي أو رفض نهائي
- تكتب سبب الرفض
- المستخدم ياخد نقاط مخالفة

---

## 2️⃣ 💵 مراجعة الإيداعات

**وظيفته:**
- يعرض لك طلبات الإيداع (المستخدمين اللي عاوزين يشحنوا فلوس)
- تقدر تقبل أو ترفض كل طلب

**كيف يشتغل:**
1. تضغط على الزر
2. يظهر لك عدد طلبات الإيداع المعلقة
3. لكل طلب يظهر:
   - اسم المستخدم
   - المبلغ
   - طريقة الإيداع (Binance ID أو محفظة)
   - صورة التحويل
   - زرارين: ✅ قبول | ❌ رفض

**مثال:**
```
💵 عدد الإيداعات المعلقة: 3

🆔 رقم الطلب: 45
👤 المستخدم: @ahmed
💰 المبلغ: 10 USDT
📍 الطريقة: Binance Pay ID
🆔 Binance ID: 123456789
⏰ وقت التحويل: 14:30
📸 [صورة التحويل]

[✅ قبول] [❌ رفض]
```

**لما تقبل:**
- المبلغ يتضاف لرصيد المستخدم
- يوصله إشعار بالقبول

**لما ترفض:**
- تكتب سبب الرفض
- المستخدم يستلم إشعار بالرفض

---

## 3️⃣ 💸 مراجعة السحوبات

**وظيفته:**
- يعرض لك طلبات السحب (المستخدمين اللي عاوزين يسحبوا فلوس)
- تقدر تقبل أو ترفض كل طلب

**كيف يشتغل:**
1. تضغط على الزر
2. يظهر لك عدد طلبات السحب المعلقة
3. لكل طلب يظهر:
   - اسم المستخدم
   - المبلغ
   - طريقة السحب (Binance ID أو محفظة)
   - عنوان المحفظة أو Binance ID
   - زرارين: ✅ تم الإرسال | ❌ رفض

**مثال:**
```
💸 عدد السحوبات المعلقة: 2

🆔 رقم الطلب: 78
👤 المستخدم: @sara
💰 المبلغ: 5 USDT
📍 الطريقة: Binance Pay ID
🆔 Binance Pay ID: 987654321

[✅ تم الإرسال] [❌ رفض]
```

**لما تقبل:**
- تحول الفلوس للمستخدم
- تضغط "تم الإرسال"
- المبلغ يتخصم من رصيده
- يوصله إشعار

**لما ترفض:**
- تكتب سبب الرفض
- المبلغ يرجع لرصيده
- يوصله إشعار بالرفض

---

## 4️⃣ 🔍 البحث عن مستخدم

**وظيفته:**
- تبحث عن أي مستخدم بالـ ID أو اليوزرنيم
- تشوف معلوماته وتتحكم فيه

**كيف يشتغل:**
1. تضغط على الزر
2. تكتب ID المستخدم أو اليوزرنيم
3. يظهر لك:
   - معلومات المستخدم
   - رصيده
   - حالته (نشط/محظور)
   - أزرار التحكم

**مثال:**
```
أرسل ID المستخدم أو اليوزرنيم:
> 123456789

👤 معلومات المستخدم:

🆔 ID: 123456789
👤 اليوزرنيم: @ahmed
💰 المحفظة: 15.50 USDT
🌐 اللغة: ar
🟢 نشط
📅 تاريخ التسجيل: 2026-01-15

[💰 تعديل المحفظة] [🚫 حظر]
```

**الأزرار المتاحة:**
- **💰 تعديل المحفظة**: تغير رصيده
- **🚫 حظر**: تحظر المستخدم
- **✅ إلغاء الحظر**: ترفع الحظر عنه

---

## 5️⃣ ✏️ تعديل نص الدعم

**وظيفته:**
- تغير رسالة الدعم اللي تظهر للمستخدمين
- تقدر تعدلها لكل لغة (عربي، إنجليزي، روسي)

**كيف يشتغل:**
1. تضغط على الزر
2. تختار اللغة
3. يظهر لك النص الحالي
4. تكتب النص الجديد

**مثال:**
```
✏️ تعديل نص الدعم

اختر اللغة:
[🇸🇦 العربية] [🇬🇧 English] [🇷🇺 Русский]

> تختار العربية

النص الحالي:
"للدعم تواصل معنا على @support"

أرسل النص الجديد:
> "للدعم راسلنا على @newsupport أو اتصل بنا"

✅ تم تحديث نص الدعم للغة العربية
```

## 6️⃣ 🔧 تغيير الحد الأقصى للأشخاص

**وظيفته:**
- تحدد أقصى عدد أشخاص يمكن طلبهم في مهمة واحدة
- مثلاً: لو حطيت 100، المستخدم ما يقدرش يطلب أكثر من 100 شخص

**كيف يشتغل:**
1. تضغط على الزر
2. يظهر لك الحد الحالي
3. تكتب الحد الجديد

**مثال:**
```
🔧 تغيير الحد الأقصى للأشخاص المطلوبين

الحد الحالي: 50 أشخاص

أرسل الحد الأقصى الجديد:
> 100

✅ تم تحديث الحد الأقصى للأشخاص المطلوبين إلى: 100
```

## 7️⃣ 📝 تغيير حد المهام للمستخدم

**وظيفته:**
- تحدد كم مهمة نشطة يقدر المستخدم يعملها في نفس الوقت
- مثلاً: لو حطيت 5، المستخدم يقدر يعمل 5 مهام نشطة بس

**كيف يشتغل:**
1. تضغط على الزر
2. يظهر لك الحد الحالي
3. تكتب الحد الجديد

**مثال:**
```
📝 تغيير حد المهام للمستخدم

الحد الحالي: 3 مهام

أرسل الحد الأقصى الجديد:
> 5

✅ تم تحديث حد المهام للمستخدم إلى: 5
```

## 8️⃣ ⏱️ تغيير وقت المهلة

**وظيفته:**
- تحدد كم دقيقة المستخدم عنده عشان يكمل المهمة
- لو ما كملهاش في الوقت ده، المهمة تلغى تلقائياً

**كيف يشتغل:**
1. تضغط على الزر
2. يظهر لك الوقت الحالي
3. تكتب الوقت الجديد بالدقائق

**مثال:**
```
⏱️ تغيير وقت المهلة للمهام

⏰ الوقت الحالي: 30 دقيقة

أرسل الوقت الجديد بالدقائق:
> 60

✅ تم تحديث وقت المهلة إلى: 60 دقيقة
```

1. تضغط على الزر

**مثال:**
```

⏰ المهلة الحالية: 15 دقيقة

أرسل المهلة الجديدة بالدقائق:
```

## 1️⃣3️⃣ 📢 رسالة جماعية

**وظيفته:**
- ترسل رسالة لكل المستخدمين أو مجموعة معينة
- مفيدة للإعلانات أو التحديثات

**كيف يشتغل:**
1. تضغط على الزر
2. تختار الفئة المستهدفة:
   - كل المستخدمين
   - المستخدمين النشطين فقط
   - المستخدمين المحظورين
3. تكتب الرسالة
4. تأكد الإرسال

**مثال:**
```
📢 رسالة جماعية

اختر الفئة المستهدفة:
[👥 الكل] [✅ النشطين] [🚫 المحظورين]

> تختار "الكل"

أرسل الرسالة:
> "🎉 تحديث جديد! تم إضافة ميزات جديدة للبوت"

تأكيد الإرسال؟
[✅ إرسال] [❌ إلغاء]

> تضغط "إرسال"

📤 جاري الإرسال...
✅ تم إرسال الرسالة إلى 1,234 مستخدم
```

   - اسم المستخدم
   - سبب الحظر

## 💡 نصائح للأدمن

### ✅ اعمل كده:
1. **راجع الإثباتات يومياً** - عشان المستخدمين ما يستنوش كتير
2. **راجع الإيداعات والسحوبات بسرعة** - عشان الناس تثق في البوت
3. **اقرأ الاستئنافات كويس** - ممكن يكون في ناس مظلومة
4. **استخدم الإحصائيات** - عشان تعرف البوت ماشي إزاي

### ❌ ما تعملش كده:
1. **ما ترفضش إثباتات صح** - ده ظلم
2. **ما تحظرش ناس بدون سبب** - استخدم نظام المخالفات
3. **ما تغيرش الإعدادات كتير** - ممكن تخرب البوت
4. **ما تنسىش تراجع الطلبات** - الناس مستنية
