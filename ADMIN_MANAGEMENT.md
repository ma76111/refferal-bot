## 📋 كيفية الاستخدام

### 1️⃣ الوصول إلى إدارة الأدمنز
```
الأدمن الرئيسي → لوحة تحكم الأدمن → 👥 إدارة الأدمنز
```

### 2️⃣ إضافة أدمن ثانوي
1. اضغط على زر **➕ إضافة أدمن**
2. أرسل ID المستخدم (يجب أن يكون قد استخدم البوت مرة واحدة)
3. سيتم إضافته فوراً وإرسال إشعار له

**مثال:**
```
أرسل: 123456789
✅ تم إضافة الأدمن بنجاح!
```

### 3️⃣ إزالة أدمن ثانوي
1. اضغط على زر **➖ إزالة أدمن**
2. أرسل ID الأدمن الذي تريد إزالته
3. سيتم إزالته فوراً وإرسال إشعار له

**مثال:**
```
أرسل: 123456789
✅ تم إزالة الأدمن بنجاح!
```

### 4️⃣ عرض قائمة الأدمنز
اضغط على زر **📋 عرض الأدمنز** لرؤية:
- 👑 الأدمن الرئيسي
- 📋 قائمة جميع الأدمنز الثانويين
- 🟢 الحالة (نشط/معطل)
- 📅 تاريخ الإضافة

## 🗄️ قاعدة البيانات

### جدول `admins`
```sql
CREATE TABLE admins (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  telegram_id INTEGER UNIQUE NOT NULL,
  username TEXT,
  added_by INTEGER NOT NULL,
  role TEXT DEFAULT 'secondary',
  permissions TEXT DEFAULT 'all',
  is_active INTEGER DEFAULT 1,
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  FOREIGN KEY (added_by) REFERENCES users(id)
);
```

### الحقول
- `telegram_id`: معرف تيليجرام للأدمن
- `username`: اليوزرنيم (اختياري)
- `added_by`: ID الأدمن الذي أضافه
- `role`: الدور (secondary فقط)
- `permissions`: الصلاحيات (all حالياً)
- `is_active`: الحالة (1 = نشط، 0 = معطل)
- `created_at`: تاريخ الإضافة

## 🔧 الدوال البرمجية

### `Admin.isMainAdmin(telegramId)`
التحقق من أن المستخدم هو الأدمن الرئيسي
```javascript
if (Admin.isMainAdmin(userId)) {
  // هذا هو الأدمن الرئيسي
}
```

### `Admin.isAdmin(telegramId)`
التحقق من أن المستخدم أدمن (رئيسي أو ثانوي)
```javascript
const isAdmin = await Admin.isAdmin(userId);
if (isAdmin) {
  // يمكنه الوصول لصلاحيات الأدمن
}
```

### `Admin.addAdmin(telegramId, username, addedBy)`
إضافة أدمن ثانوي (فقط الأدمن الرئيسي)
```javascript
await Admin.addAdmin(123456789, 'username', mainAdminId);
```

### `Admin.removeAdmin(telegramId, removedBy)`
إزالة أدمن ثانوي (فقط الأدمن الرئيسي)
```javascript
await Admin.removeAdmin(123456789, mainAdminId);
```

### `Admin.getAllAdmins()`
الحصول على قائمة جميع الأدمنز الثانويين
```javascript
const admins = await Admin.getAllAdmins();
```

### `Admin.getAllAdminIds()`
الحصول على جميع IDs الأدمنز (رئيسي + ثانويين نشطين)
```javascript
const adminIds = await Admin.getAllAdminIds();
// [8339087985, 123456789, 987654321]
```

## 🛡️ الحماية الأمنية

### 1. حماية الأدمن الرئيسي
```javascript
// في معالج الحظر
if (Admin.isMainAdmin(userId)) {
  await bot.answerCallbackQuery(query.id, { 
    text: '❌ لا يمكن حظر الأدمن الرئيسي', 
    show_alert: true 
  });
  return;
}
```

### 2. التحقق من الصلاحيات
```javascript
// التحقق من أن المستخدم أدمن (رئيسي أو ثانوي)
const isAdmin = await Admin.isAdmin(msg.from.id);
if (!isAdmin) {
  await bot.sendMessage(chatId, '❌ غير مصرح لك بهذا الأمر');
  return;
}

// فقط الأدمن الرئيسي يمكنه إضافة أدمنز
if (!Admin.isMainAdmin(msg.from.id)) {
  await bot.sendMessage(chatId, '❌ هذه الميزة متاحة فقط للأدمن الرئيسي');
  return;
}
```

### 3. منع الإضافة المكررة
```javascript
try {
  await Admin.addAdmin(userId, username, mainAdminId);
} catch (error) {
  if (error.message === 'Admin already exists') {
    // الأدمن موجود بالفعل
  }
}
```

### 4. Middleware للحظر
```javascript
// في middleware - تجاهل جميع الأدمنز (رئيسي وثانوي)
const isAdmin = await Admin.isAdmin(msg.from.id);
if (isAdmin) {
  return; // السماح للأدمن بالمرور
}
```

## ⚠️ ملاحظات مهمة

1. **الأدمن الرئيسي محمي بالكامل** - لا يمكن حظره أو إزالته بأي طريقة
2. **المستخدم يجب أن يستخدم البوت أولاً** - لا يمكن إضافة أدمن غير موجود في قاعدة البيانات
3. **الإشعارات التلقائية** - يتم إرسال إشعار للأدمن عند الإضافة/الإزالة
4. **الصلاحيات الكاملة** - حالياً جميع الأدمنز الثانويين لديهم نفس الصلاحيات
5. **جميع فحوصات الأدمن تستخدم `Admin.isAdmin()`** - للتحقق من الأدمنز الرئيسي والثانويين
6. **الأدمنز الثانويين محميين من الحظر** - يتم تجاهلهم في middleware الحظر
