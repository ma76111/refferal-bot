        ┌───────────────────────────────────┐
        │  إعادة التأهيل التلقائية:        │
        │  • تقليل نقطة واحدة من كل مستخدم │
        │  • إذا وصلت لـ 0 = رفع الحظر     │
        └───────────────────────────────────┘
                            │
                            ▼
        ┌───────────────────────────────────┐
        │  UPDATE violations                │
        │  SET points = points - 1          │
        │  WHERE points > 0                 │
        └───────────────────────────────────┘

  المخالفة: بلاغ من مستخدم

🎯 الخلاصة
═══════════════════════════════════════════════════════════════

النظام = نقاط + وقت + إعادة تأهيل

✅ كل مخالفة = نقاط
✅ النقاط تحدد العقوبة
✅ العقوبة تطبق تلقائياً
✅ النقاط تقل مع الوقت
✅ الحظر يرفع تلقائياً

الهدف: تشجيع السلوك الجيد ومنع الإساءة
