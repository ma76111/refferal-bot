# 🚫 نظام الحظر - شرح بسيط

## الفكرة العامة

تخيل نظام الحظر زي **نظام النقاط في المدرسة**:
- كل ما تعمل حاجة غلط → تاخد نقاط سلبية
- النقاط تتجمع
- لما توصل لحد معين → تتعاقب
- مع الوقت النقاط تقل → تتحسن حالتك

---

## 📊 أنواع المخالفات

### 1. رفض الإثبات (1 نقطة)
**متى يحصل؟**
- أنت بتنفذ مهمة
- بتبعت إثبات إنك عملتها
- الأدمن يشوف الإثبات ويقول "لا، مش صح"
- تاخد نقطة واحدة

**مثال:**
```
المهمة: اشترك في قناة تليجرام
الإثبات: صورة من قناة تانية
النتيجة: رفض → 1 نقطة
```

---

### 2. رفض نهائي (2 نقطة)
**متى يحصل؟**
- نفس الموقف فوق
- بس الأدمن يقول "رفض نهائي، مش هديك فرصة تانية"
- تاخد نقطتين

**مثال:**
```
المهمة: اشترك في قناة
الإثبات: صورة مفبركة أو من قناة تانية خالص
النتيجة: رفض نهائي → 2 نقطة
```

---

### 3. بلاغ من مستخدم (3 نقاط)
**متى يحصل؟**
- مستخدم تاني يبلغ عنك
- يقول إنك بتغش أو بتعمل حاجة غلط
- تاخد 3 نقاط

**مثال:**
```
أنت: عملت مهمة وطلبت من صاحبك يعملها بدالك
صاحب المهمة: اكتشف وبلغ عنك
النتيجة: بلاغ → 3 نقاط
```

---

### 4. حظر يدوي من الأدمن (10 نقاط)
**متى يحصل؟**
- الأدمن يحظرك بنفسه مباشرة
- يعني عملت حاجة وحشة جداً
- تاخد 10 نقاط مرة واحدة

**مثال:**
```
أنت: بتحاول تخترق البوت أو بتشتم الناس
الأدمن: يحظرك مباشرة
النتيجة: حظر يدوي → 10 نقاط
```

---

## 🎯 مستويات العقوبة

### المستوى 1: تحذير (0-2 نقطة)
**إيه اللي يحصل؟**
- ولا حاجة! 😊
- بس تحذير إنك لو كملت هتتعاقب
- تقدر تستخدم البوت عادي

**مثال:**
```
النقاط: 1 نقطة
الرسالة: "⚠️ تحذير: لديك مخالفة واحدة"
التأثير: لا شيء، استمر بحذر
```

---

### المستوى 2: تقييد (3-4 نقاط)
**إيه اللي يحصل؟**
- تتمنع من إضافة مهام جديدة
- تتمنع من تنفيذ مهام
- بس تقدر تسحب فلوسك
- المدة: 24 ساعة

**مثال:**
```
النقاط: 3 نقاط
الرسالة: "⏸️ أنت مقيد لمدة 24 ساعة"
التأثير: 
  ❌ لا يمكنك إضافة مهام
  ❌ لا يمكنك تنفيذ مهام
  ✅ يمكنك السحب فقط
```

---

### المستوى 3: حظر مؤقت (5-9 نقاط)
**إيه اللي يحصل؟**
- تتحظر من البوت كله
- ما تقدرش تستخدمه نهائياً
- المدة: 7 أيام
- تقدر تقدم استئناف

**مثال:**
```
النقاط: 5 نقاط
الرسالة: "🚫 أنت محظور لمدة 7 أيام"
التأثير: 
  ❌ لا يمكنك استخدام البوت نهائياً
  ✅ يمكنك تقديم استئناف
```

---

### المستوى 4: حظر دائم (10+ نقاط)
**إيه اللي يحصل؟**
- تتحظر للأبد
- ما تقدرش تستخدم البوت أبداً
- بس تقدر تقدم استئناف والأدمن يراجعه

**مثال:**
```
النقاط: 10 نقاط
الرسالة: "🔴 أنت محظور بشكل دائم"
التأثير: 
  ❌ لا يمكنك استخدام البوت أبداً
  ✅ يمكنك تقديم استئناف (الأدمن يقرر)
```

---

## 🔄 إعادة التأهيل التلقائية

**إيه ده؟**
- كل 24 ساعة، النظام يشيل نقطة واحدة من كل الناس
- يعني لو عندك 5 نقاط، بعد 5 أيام هتبقى 0
- لما توصل 0 → الحظر يترفع تلقائياً

**مثال عملي:**

```
اليوم 1:
  عملت مخالفة → 5 نقاط
  العقوبة: حظر 7 أيام 🚫

اليوم 2:
  النظام شال نقطة → 4 نقاط
  العقوبة: تقييد 24 ساعة ⏸️

اليوم 3:
  النظام شال نقطة → 3 نقاط
  العقوبة: تقييد 24 ساعة ⏸️

اليوم 4:
  النظام شال نقطة → 2 نقطة
  العقوبة: تحذير فقط ⚠️

اليوم 5:
  النظام شال نقطة → 1 نقطة
  العقوبة: تحذير فقط ⚠️

اليوم 6:
  النظام شال نقطة → 0 نقطة
  العقوبة: لا شيء ✅ نظيف تماماً!
```

## 📱 كيف يشتغل في البوت؟

### السيناريو الكامل:

**1. أنت بتستخدم البوت عادي**
```
أنت: /start
البوت: مرحباً! اختر من القائمة
```

**2. تنفذ مهمة وتبعت إثبات غلط**
```
أنت: [صورة إثبات غلط]
الأدمن: يرفض الإثبات
النظام: يضيف 1 نقطة
البوت: "⚠️ تم رفض إثباتك. لديك الآن 1 نقطة مخالفة"
```

**3. تعمل نفس الغلط تاني**
```
أنت: [صورة إثبات غلط مرة تانية]
الأدمن: يرفض الإثبات
النظام: يضيف 1 نقطة → الإجمالي 2
البوت: "⚠️ تم رفض إثباتك. لديك الآن 2 نقطة مخالفة"
```

**4. حد يبلغ عنك**
```
مستخدم تاني: يبلغ عنك
النظام: يضيف 3 نقاط → الإجمالي 5
البوت: "🚫 أنت محظور لمدة 7 أيام بسبب تجاوز حد المخالفات"
```

**5. تحاول تستخدم البوت**
```
أنت: /start
البوت: "🚫 أنت محظور حتى [التاريخ]. السبب: تجاوز حد المخالفات"
```

**6. بعد أسبوع**
```
النظام: شال نقطة كل يوم
النقاط الآن: 0
البوت: "✅ تم رفع الحظر عنك. يمكنك استخدام البوت الآن"
```

## 🛡️ الحماية

### 1. الأدمن محمي
```
لو حاولت تبلغ عن الأدمن → مش هيحصل حاجة
الأدمن ما يتحظرش أبداً
```

### 2. كل حاجة مسجلة
```
كل مخالفة مسجلة مع:
- التاريخ والوقت
- السبب
- مين اللي عملها
- كام نقطة
```

### 3. الاستئناف
```
لو اتحظرت ظلم → تقدر تقدم استئناف
الأدمن يراجعه ويقرر
لو موافق → يرفع الحظر
لو رافض → تفضل محظور
```

---

## 💡 نصائح عشان ما تتحظرش

### ✅ اعمل كده:
1. **نفذ المهام صح**
   - اقرأ التعليمات كويس
   - ابعت إثبات صحيح

2. **ما تغشش**
   - ما تطلبش من حد يعمل المهمة بدالك
   - ما تفبركش الصور

3. **احترم الناس**
   - ما تشتمش
   - ما تحاولش تنصب على حد

4. **لو اتحظرت**
   - استنى الحظر يخلص
   - أو قدم استئناف لو مظلوم

### ❌ ما تعملش كده:
1. **ما تبعتش إثبات غلط**
   - صور من قنوات تانية
   - صور مفبركة

2. **ما تحاولش تغش**
   - تعمل حسابات وهمية
   - تطلب من ناس تعمل المهام بدالك

3. **ما تسيءش للناس**
   - شتايم
   - تهديدات
   - محاولات نصب
