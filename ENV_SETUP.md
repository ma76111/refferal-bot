# ⚙️ إعداد ملف البيئة (.env)

## 🔴 مهم جداً: لا ترفع ملف .env على GitHub!

---

## 📝 خطوات الإعداد

### على الموبايل (Termux):

بعد تحميل المشروع، اكتب:

```bash
cd ~/refferal-bot
nano .env
```

ثم انسخ والصق هذا المحتوى:

```env
# توكن البوت من @BotFather
BOT_TOKEN=ضع_توكن_البوت_هنا

# ID الأدمن الرئيسي
ADMIN_IDS=8339087985

# مفاتيح Binance API (اختياري)
BINANCE_API_KEY=ضع_مفتاح_بينانس_هنا
BINANCE_API_SECRET=ضع_سر_بينانس_هنا
```

**للحفظ:**
- اضغط `Ctrl + X`
- اضغط `Y`
- اضغط `Enter`

---

## 🔑 كيف تحصل على التوكن؟

### 1. توكن البوت:
1. افتح تليجرام
2. ابحث عن `@BotFather`
3. أرسل `/newbot` أو `/mybots`
4. انسخ التوكن

### 2. مفاتيح Binance (اختياري):
1. سجل دخول على Binance
2. اذهب إلى API Management
3. أنشئ API Key جديد
4. انسخ API Key و Secret Key

## 🔒 أمان المعلومات

⚠️ **لا تشارك هذه المعلومات مع أحد:**
- توكن البوت
- مفاتيح Binance
- محتوى ملف .env

⚠️ **إذا تسرب التوكن:**
1. اذهب إلى @BotFather
2. أرسل `/mybots`
3. اختر البوت
4. اضغط "Revoke Token"
5. احصل على توكن جديد
6. حدّث ملف .env

---

## 🆘 حل المشاكل

### المشكلة: البوت لا يعمل بعد إعداد .env
```bash
# تحقق من وجود الملف
ls -la .env

# تحقق من محتوى الملف
cat .env

# تأكد من عدم وجود مسافات زائدة
nano .env
```

### المشكلة: خطأ "BOT_TOKEN is not defined"
- تأكد من كتابة `BOT_TOKEN=` بدون مسافات
- تأكد من وجود التوكن بعد علامة `=`
- لا تضع مسافات قبل أو بعد `=`
