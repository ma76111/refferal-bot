# Telegram Referral Exchange Bot

بوت تليجرام لتبادل الإحالات والمهام المدفوعة

## المميزات

- ✅ نظام مهام مدفوعة (USDT)
- ✅ نظام تبادل الإحالات
- ✅ نظام إيداع وسحب USDT
- ✅ نظام مراجعة متقدم مع خيارات رفض
- ✅ نظام إبلاغ عن المخالفين
- ✅ حماية من السبام
- ✅ دعم 3 لغات (عربي، إنجليزي، روسي)

## التثبيت

1. تثبيت المتطلبات:
```bash
npm install
```

2. إعداد ملف `.env`:
```env
BOT_TOKEN=your_bot_token_here
ADMIN_IDS=123456789,987654321
```

3. تشغيل البوت:
```bash
node index.js
```

## إعادة تعيين البوت

لإعادة تعيين البوت بالكامل (حذف جميع البيانات):
```bash
node reset_bot.js
```

⚠️ **تحذير**: هذا سيحذف جميع البيانات نهائياً!
