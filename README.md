# 🤖 Referral Bot - بوت تبادل الإحالات (الإصدار 2.0)

## 🎉 الحالة: ✅ **جاهز للإنتاج**
## 🌟 التقييم الأمني: **9.5/10**

بوت تليجرام متقدم لتبادل الإحالات والمهام المدفوعة مع نظام مراجعة متطور ودعم متعدد اللغات.

---

## ✨ المميزات الرئيسية

### 🎯 نظام المهام
- **مهام مدفوعة**: ادفع للمستخدمين مقابل الإحالات
- **تبادل الإحالات**: نفذ مهام الآخرين للحصول على إحالات مجانية
- **أنواع إثبات متعددة**: نص، صور، أو كليهما
- **مهلة زمنية قابلة للتعديل**: تحكم في وقت تنفيذ المهام

### 📝 نظام المراجعة المتقدم
- **رفض مع فرصة ثانية**: امنح المستخدمين فرصة لتحسين الإثبات
- **رفض نهائي**: مع إمكانية الإبلاغ عن صاحب المهمة
- **مهلة التحسين**: 30 دقيقة افتراضياً (قابلة للتعديل)
- **حذف تلقائي للصور**: توفير المساحة بعد المراجعة

### 🚨 نظام المخالفات والحماية المتقدم
- **نظام النقاط**: تراكم نقاط المخالفات
- **حظر تلقائي**: بعد 100 نقطة
- **تقييدات تلقائية**: منع إضافة مهام أو تنفيذها
- **إعادة تأهيل**: تقليل النقاط للمستخدمين الملتزمين
- **حماية من الظلم**: نظام عادل للمخالفات

### ⭐ نظام التقييمات (جديد)
- **تقييم المستخدمين**: من 1-5 نجوم
- **تعليقات اختيارية**: إضافة ملاحظات
- **متوسط التقييم**: عرض تقييم المستخدم
- **توزيع التقييمات**: إحصائيات مفصلة
- **آخر التقييمات**: عرض آخر 5 تقييمات

### 📢 نظام الرسائل الجماعية (جديد)
- **إرسال لجميع المستخدمين**: رسالة واحدة للجميع
- **استهداف محدد**: نشطون، محظورون، أو الجميع
- **معاينة قبل الإرسال**: تأكيد الرسالة
- **إحصائيات الإرسال**: نجاح/فشل
- **سجل الرسائل**: عرض الرسائل السابقة

### 📝 نظام الاستئناف (جديد)
- **استئناف الحظر**: السماح للمحظورين بالاستئناف
- **مراجعة الاستئنافات**: من الأدمن
- **رفع الحظر التلقائي**: عند القبول
- **منع التكرار**: استئناف واحد معلق فقط

### 📊 نظام الإحصائيات (جديد)
- **لوحة شاملة**: إحصائيات كاملة للأدمن
- **إحصائيات المستخدمين**: عدد، نشطون، محظورون
- **إحصائيات المهام**: مدفوعة، تبادل، مكتملة
- **إحصائيات المالية**: إيداعات، سحوبات، أرباح
- **أفضل 10 مستخدمين**: حسب الرصيد

### 💰 نظام المدفوعات
- **إيداع USDT**: عبر Binance Pay ID أو عنوان المحفظة
- **سحب USDT**: شبكة BSC
- **حد أدنى للسحب**: قابل للتعديل
- **مراجعة يدوية**: للأمان والحماية

### 🌐 دعم متعدد اللغات
- العربية 🇸🇦
- الإنجليزية 🇬🇧
- الروسية 🇷🇺

### 👥 ثلاثة أدوار

3. **Admin**: يدير البوت ويراجع العمليات

## 🚀 التثبيت

### المتطلبات
- Node.js 18+ 
- npm أو yarn
- حساب Telegram Bot (من [@BotFather](https://t.me/BotFather))

### خطوات التثبيت

1. **استنساخ المشروع**
```bash
git clone https://github.com/YOUR_USERNAME/refferalbot.git
cd refferalbot
```

2. **تثبيت المكتبات**
```bash
npm install
```

3. **إعداد البيئة**
```bash
cp .env.example .env
```

4. **تعديل ملف `.env`**
```env
BOT_TOKEN=your_bot_token_here
ADMIN_IDS=123456789,987654321
DATABASE_PATH=./bot.db
```

5. **تشغيل البوت**
```bash
node index.js
```
