# 🛡️ نظام المخالفات والعقوبات المتقدم

## 📋 نظرة عامة

نظام متقدم لإدارة المخالفات والعقوبات في البوت، يوفر نهجاً عادلاً ومتدرجاً للتعامل مع المخالفات.

---

## 🎯 المميزات الرئيسية

### ✅ **نظام النقاط**
- كل مخالفة لها عدد محدد من النقاط
- النقاط تتراكم مع كل مخالفة
- العقوبة تعتمد على إجمالي النقاط

### ⚠️ **التحذيرات المتدرجة**
- تحذير أول (3-4 نقاط)
- تحذير ثاني + تقييد (5-7 نقاط)
- حظر مؤقت 3 أيام (8-10 نقاط)
- حظر مؤقت 7 أيام (11-14 نقاط)
- حظر دائم (15+ نقاط)

### 🔒 **الحظر المؤقت**
- حظر لمدة محددة (أيام)
- يتم رفع الحظر تلقائياً
- مهمة دورية تفحص الحظر المنتهي

### 🚫 **التقييدات الجزئية**
- منع إضافة مهام
- منع السحب
- منع الإبلاغ
- تقليل عدد المهام
- زيادة وقت الانتظار

### 🔄 **إعادة التأهيل**
- تقليل نقطة واحدة كل 30 يوم
- تشجيع السلوك الجيد
- فرصة للعودة

## 📊 أنواع المخالفات

| المخالفة | النقاط | الوصف |
|---------|--------|-------|
| `REPORT_RECEIVED` | 1 | تلقي إبلاغ من مستخدم |
| `SUBMISSION_REJECTED` | 2 | رفض إثبات نهائي |
| `SPAM_REPORTS` | 3 | إرسال إبلاغات متكررة |
| `ABUSIVE_BEHAVIOR` | 3 | سلوك مسيء |
| `FAKE_TASK` | 4 | إنشاء مهمة وهمية |
| `FRAUD_ATTEMPT` | 5 | محاولة احتيال |
| `MULTIPLE_ACCOUNTS` | 10 | حسابات متعددة |

---

## 🎚️ مستويات العقوبات

### 1️⃣ **تحذير أول** (3-4 نقاط)
- رسالة تحذيرية فقط
- لا إجراءات إضافية
- تنبيه للمستخدم

### 2️⃣ **تحذير ثاني + تقييد** (5-7 نقاط)
- رسالة تحذير
- منع إضافة مهام لمدة 24 ساعة
- تحذير من الحظر المؤقت

### 3️⃣ **حظر مؤقت 3 أيام** (8-10 نقاط)
- حظر كامل من البوت
- مدة: 3 أيام
- يتم رفع الحظر تلقائياً

### 4️⃣ **حظر مؤقت 7 أيام** (11-14 نقاط)
- حظر كامل من البوت
- مدة: 7 أيام
- تحذير من الحظر الدائم

### 5️⃣ **حظر دائم** (15+ نقاط)
- حظر دائم من البوت
- يمكن الاستئناف
- يحتاج موافقة الأدمن للرفع

## 🔧 الاستخدام

### إضافة مخالفة

```javascript
import ViolationSystem from './utils/violationSystem.js';

// إضافة مخالفة
const result = await ViolationSystem.addViolation(
  userId,
  'SPAM_REPORTS',
  'إرسال 5 إبلاغات في 5 دقائق'
);

console.log(`النقاط الإجمالية: ${result.totalPoints}`);
```

### التحقق من حالة المستخدم

```javascript
const status = await ViolationSystem.checkUserStatus(userId);

if (!status.allowed) {
  console.log(`المستخدم محظور: ${status.reason}`);
  console.log(`الوقت المتبقي: ${status.remaining}`);
}
```

### الحصول على رسالة العقوبة

```javascript
const penalty = { action: 'temp_ban', duration: 3 };
const message = ViolationSystem.getPenaltyMessage(
  penalty,
  totalPoints,
  'ar'
);

await bot.sendMessage(userId, message);
```

## 🗄️ قاعدة البيانات

### جدول المخالفات (`violations`)
```sql
CREATE TABLE violations (
  id INTEGER PRIMARY KEY,
  user_id INTEGER,
  type TEXT,
  points INTEGER,
  reason TEXT,
  created_at DATETIME,
  expires_at DATETIME,
  status TEXT
);
```

### جدول الحظر (`bans`)
```sql
CREATE TABLE bans (
  id INTEGER PRIMARY KEY,
  user_id INTEGER,
  type TEXT,
  duration INTEGER,
  reason TEXT,
  banned_by INTEGER,
  start_date DATETIME,
  end_date DATETIME,
  status TEXT
);
```

### جدول التقييدات (`restrictions`)
```sql
CREATE TABLE restrictions (
  id INTEGER PRIMARY KEY,
  user_id INTEGER,
  type TEXT,
  duration INTEGER,
  start_date DATETIME,
  end_date DATETIME,
  reason TEXT,
  status TEXT
);
```

## ⏰ المهام الدورية

### 1. رفع الحظر المؤقت (كل 10 دقائق)
```javascript
setInterval(async () => {
  await Ban.liftExpiredBans();
}, 10 * 60 * 1000);
```

### 2. رفع التقييدات (كل 10 دقائق)
```javascript
setInterval(async () => {
  await Restriction.liftExpiredRestrictions();
}, 10 * 60 * 1000);
```

### 3. إعادة التأهيل (كل 24 ساعة)
```javascript
setInterval(async () => {
  await Violation.rehabilitateUsers();
}, 24 * 60 * 60 * 1000);
```
